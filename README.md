# TMT-Net

**Теория Метрики Транзакций**  
Одна формула: **t ≡ A** (Время = Действие)(реализуется как A = E * t)

## Что это
Нейросеть, где время течет только когда совершаются транзакции.
Нейроны имеют массу, горизонт событий и голографическую память.
При переполнении горизонта происходит коллапс с излучением Хокинга.
